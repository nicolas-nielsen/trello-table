## Getting Started

Install dependecies:

```bash
npm install
```

Run the server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Dependencies

- @emotion/react
- @emotion/styled
- @mui/icons-material
- @mui/material
- uuid
